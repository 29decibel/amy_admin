# AmyAdmin

a simple rails_admin featuring rich javascript client + robust server api

## Installation

Add this line to your application's Gemfile:

    gem 'amy_admin'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install amy_admin

## Usage

```ruby
AmyAdmin.register 'topics' do
  index do
  end
end
```
